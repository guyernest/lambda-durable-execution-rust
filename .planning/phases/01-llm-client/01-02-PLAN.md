<objective>
Implement Anthropic and OpenAI message transformers with the MessageTransformer trait and TransformerRegistry.

Purpose: These transformers are the core provider abstraction -- they convert between the unified LLMInvocation/LLMResponse types and each provider's wire format. The unified FunctionCall extraction (LLM-07) is the key deliverable enabling the agent loop to work regardless of provider.

Output: Two complete transformers and a registry, tested with unit tests for request building, response parsing, and function call extraction.
</objective>
