**AgentConfig model (per D-04, D-05):**
```
AgentConfig: a
  .model({
    id: a.id().required(),
    organizationId: a.string().required(),
    name: a.string().required(),
    displayName: a.string(),
    description: a.string(),
    status: a.enum(['active', 'inactive']),

    // LLM Configuration
    modelId: a.string().required(),        // References LLMModel.id
    temperature: a.float().default(0.7),
    maxTokens: a.integer().default(4096),
    maxIterations: a.integer().default(10),

    // Agent Instructions
    instructions: a.string().required(),    // System prompt

    // MCP Server Selection (per D-05: from pmcp-run registry only)
    mcpServerIds: a.string().array(),       // Array of McpServer.id references

    // Parameters
    parameters: a.json(),                   // Arbitrary key-value config

    // Metadata
    createdAt: a.datetime(),
    updatedAt: a.datetime(),
    createdBy: a.string(),
  })
  .secondaryIndexes((index) => [
    index('organizationId'),
    index('name'),
  ])
  .authorization((allow) => [
    allow.authenticated().to(['read', 'create', 'update', 'delete']),
  ]),
```

**AgentExecution model (per D-06, D-07, D-08, D-09, D-11):**
```
AgentExecution: a
  .model({
    id: a.id().required(),
    organizationId: a.string().required(),
    agentConfigId: a.string().required(),   // References AgentConfig.id
    agentName: a.string().required(),       // Denormalized for list display

    // Execution State (per D-09)
    status: a.enum(['pending', 'running', 'completed', 'failed']),
    input: a.string(),                      // User's input prompt
    output: a.string(),                     // Final agent response text

    // Durable Execution Bridge (per D-07)
    durableExecutionId: a.string(),         // For deep inspection via Durable Execution API

    // Token Tracking (per D-11)
    inputTokens: a.integer().default(0),
    outputTokens: a.integer().default(0),
    totalTokens: a.integer().default(0),

    // Iteration Tracking
    iterations: a.integer().default(0),
    toolsCalled: a.string().array(),        // Tool names called during execution

    // Timing
    startedAt: a.datetime(),
    completedAt: a.datetime(),
    durationMs: a.integer(),

    // Error Info
    errorMessage: a.string(),

    // Model used (denormalized for cost calculation per D-13)
    modelId: a.string(),
    modelDisplayName: a.string(),
  })
  .secondaryIndexes((index) => [
    index('agentConfigId')
      .sortKeys(['startedAt'])
      .name('byAgentConfigSortedByStartedAt')
      .queryField('listExecutionsByAgent'),
    index('organizationId')
      .sortKeys(['startedAt'])
      .name('byOrgSortedByStartedAt')
      .queryField('listExecutionsByOrg'),
    index('status'),
  ])
  .authorization((allow) => [
    allow.authenticated().to(['read', 'create', 'update', 'delete']),
  ]),
```

**LLMModel model (per D-14):**

```
LLMModel: a

    maxTokens: a.integer(),                  // Max output tokens
    contextWindow: a.integer(),              // Context window size

    isActive: a.boolean().default(true),

    // Metadata
    createdAt: a.datetime(),
    updatedAt: a.datetime(),

  .secondaryIndexes((index) => [
    index('provider'),
    index('modelId'),
  ])
  .authorization((allow) => [
    allow.authenticated().to(['read', 'create', 'update', 'delete']),
  ]),
```

```typescript
listAgentSecrets: a
  .query()

    organizationId: a.string().required(),

  .returns(a.customType({
    organizationId: a.string().required(),
    secrets: a.json().required(),

  .authorization((allow) => [allow.authenticated()])

setAgentSecret: a

deleteAgentSecret: a

  }))
  .authorization((allow) => [allow.authenticated()])
  .handler(a.handler.function(manageAgentSecrets)),
```
